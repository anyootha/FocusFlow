# FocusFlow

### JavaScript-сервис подготовки сотрудника к встрече с руководителем

---

## О проекте

FocusFlow — это внутренний веб-сервис, который помогает сотрудникам быстро и качественно подготовиться к встречам с руководителями  
(1-on-1, статус-встречи, защита проектов, обсуждение повышения).

Проект создан как MVP, ориентированный не на «красивый код», а на:
- экономию времени руководителей  
- повышение качества встреч  
- фиксацию договорённостей  
- быстрый запуск и масштабирование  

---

## Проблема

На практике:
- сотрудники приходят на встречи неподготовленными  
- разговоры уходят в абстракции  
- решения не фиксируются  
- встречи затягиваются и повторяются  

---

## Решение

Перед встречей сотрудник:
1. заполняет короткую форму  
2. получает AI-анализ своей готовности  
3. видит слабые места и рекомендации  
4. отчёт автоматически отправляется в Telegram  

Встречи становятся короче, конкретнее и результативнее.

---

## Функциональность MVP

- форма подготовки к встрече  
- анализ ответов с помощью AI  
- рекомендации по формулировкам  
- выявление рисков  
- Telegram-уведомления  
- frontend + backend на JavaScript
